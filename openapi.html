Introduction

The Centralized Accounting and Reporting Platform (CARP) supports and facilitates the submission of reports and the publication of non-confidential information according to decision 2/CMA.3, annex, chapter IV (Reporting).

Authorization

Authorization operations

Register authorisation from Party submitting request.

Register cooperative approach authorization from the Party submitting the request.
This will be performed by the secretariat when it registers the CA authorisation with the CARP.
This enables the CAs and ITMOs for interoperability.
This will create the CooperativeApproachAuthorization and the CooperativeApproach within the A6 DB.

Authorizations:
basicAuth
Request Body schema: application/json
required

The authorisation to be registered.

authorizingPartyId
required
string

ISO-3166-1 alpha-3 code of the participating Party reporting this authorization,
as per common nomenclatures.

authorisationId
required
string

Authorization Id as assigned by the participating Party reporting the authorisation.

authorisationDate
required
string <date>

Datestamp of the authorisation of the cooperative approach. A cooperative approach may be authorised multiple times. Only the latest is valid.

version
required
string

Version of the authorisation of the cooperative approach. The latest version is valid.

lastChangeDate
required
string <date>

Date of last change to the authorization.

effectiveChangeDate
string <date>

Date of last change to the authorization.

authStartDate
required
string <date>

start date of the authorization.

authEndDate
required
string <date>

end date of the authorization.

ndcStartDate
string <date>

start date of NDC period of the issuing Party.

ndcEndDate
string <date>

end date of the NDC period of the issuing Party.

isCAAuth
required
boolean

Is this authorization for a cooperative approach

object (CAAuthElements)
isMOAuth
required
boolean

Is this authorization for Mitigation Outcomes

object (MOAuthElements)
isEntityAuth
required
boolean

Is this authorization for a entities

object (EntityAuthElements)
possibleChanges
Array of strings

Where changes to the authorizaton may occur,

information on the circumstances in which such changes may occur,

and a desciption of the process for managing them

in a way that avoids double counting.

actualChanges
Array of strings

Where changes to the authorizaton have occurred,

a description of the curcumstances in which the change occurred,

and and how the changes follow the process for managing them

in a way that avoids double counting.

changeDelta
Array of strings

Where changes to the authorizaton have occurred,

a description of the specific change in respect,

to the earlier version of the authorization.

termsAndConditions
Array of strings

optionally provided terms and conditions for this authorisation

additionalInfo
Array of strings[ items <= 255 characters ]

Any additional information the Party submitted with the authorisation.

Update cooperative approach identified by caId,<br> from the registry identified by regId.

Update cooperative approach identified by caId,
from the registry identified by regId,
with attributes from the cooperative approach in the requestBody.

Authorizations:
basicAuth
path Parameters
caId
required
string
Example: CA0002

The id of the cooperative approach be updated.

Request Body schema: application/json
required

The cooperative approach authorisation to be registered.

cooperativeApproachId
required
string

Cooperative approaches under Article 6.2.

The format of the string is "CANNNN" where the "CA" is the fixed prefix, and NNNN is a four-digit integer,

that shall be incremented by one for each additional agreed cooperative approach, and added to this common nomenclature.

name
required
string

Name of the cooperative approach.

description
required
string

Description of the cooperative approach
2/CMA.3 annex, para 18(g)

authId
required
string

Authorization Id as assigned by the Party authorizing this cooperative approach.
2/CMA.3 annex, para 18(g)

startDate
required
string <date>

start date of the cooperative approach.
2/CMA.3 annex, para 18(g)

endDate
required
string <date>

end date of the cooperative approach.
2/CMA.3 annex, para 18(g)

expectedMitigationPA
required
integer

Expected mitigation of GHG per annum.
2/CMA.3 annex, para 18(g)

otherParticipatingParties
required
Array of strings

Identifiers of other participating Parties covered by authorization of cooperative approach
2/CMA.3 annex, para 18(g)

authorisedEntityIds
Array of strings

Unique identifier of entities that are authorised as per common nomenclatures,
for this cooperative approach, if applicable.
2/CMA.3 annex, para 18(g)

howNoIncreaseInGE
any

How the cooperative approach ensures there is no net increase in global emissions within and between NDC implementation periods.
2/CMA.3 annex, para. 18(h)(i)

howRobustGovernance
any

How the cooperative approach ensures environmental integrity through robust, transparent governance and the quality of mitigation outcomes.
2/CMA.3 annex, para. 18(h)(ii)

howMinRiskOfNonPerm
any

How the cooperative approach is minimizing the risk of non-permanence of mitigation across several NDC periods.
2/CMA.3 annex, para. 18(h)(iii)

howAvoidNegImpact
any

How the cooperative approach minimizes and, where possible, avoids negative environmental, economic and social impacts.
2/CMA.3 annex, para. 18(i)(i)

HowReflectParis
any

How the cooperative approach reflects the eleventh preambular paragraph of the Paris Agreement.
2/CMA.3 annex, para. 18(i)(ii)

howConsistentWithSD
any

How the cooperative approach is consistent with the sustainable development objectives of the Party, noting national prerogatives.
2/CMA.3 annex, para. 18(i)(iii)

howApplySafeguards
any

How how the cooperative approach applies any safeguards and limits set out in further guidance from the CMA.
2/CMA.3 annex, para. 18(i)(iv)

howContributeToAdapt
any

How the cooperative approach contributes resources for adaptation.
2/CMA.3 annex, para. 18(i)(v)

howDeliverOMGE
any

How the cooperative approach delivers overall mitigation in global emissions.
2/CMA.3 annex, para. 18(i)(vi)
